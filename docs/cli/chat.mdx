```bash
hexgate chat --agent example_agent
hexgate chat --list-agents
hexgate chat --agent researcher
hexgate chat --agent examples.customer_bot:agent     # module:attr spec
hexgate chat --use examples/file_agents.py --agent workspace_explorer
```

The chat command sets `HEXGATE_LOCAL_MODE=1` automatically so audit stays on
your machine. Denies and approval-required calls render as inline panels in the
terminal.
