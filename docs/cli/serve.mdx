```bash
hexgate serve examples.customer_bot:agent
hexgate serve examples.customer_bot:agent --no-auto-register
```

Takes the same `module:attr` spec as `hexgate register`. Imports the agent,
derives the manifest, auto-registers, fetches policy, and opens the WebSocket
relay so the dashboard's Playground can drive it.
